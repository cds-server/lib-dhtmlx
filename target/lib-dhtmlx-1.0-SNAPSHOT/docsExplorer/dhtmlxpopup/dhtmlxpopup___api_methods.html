dhtmlxPopup

Methods

  • Attaching content
    • attachAccordion - adds a dhtmlxAccordion object to the pop-up window
    • attachCalendar - adds a dhtmlxCalendar object to the pop-up window
    • attachForm - adds a dhtmlxForm object to the pop-up window
    • attachGrid - adds a dhtmlxGrid object to the pop-up window
    • attachHTML - adds the HTML markup to the pop-up window
    • attachLayout - adds a dhtmlxLayout object to the pop-up window
    • attachList - adds a list-like structure to the pop-up window
    • attachObject - adds a DOM element to the pop-up window
    • attachTabbar - adds a dhtmlxTabbar object to the pop-up window
    • attachTree - adds a dhtmlxTree object to the pop-up window
  • Clearing content
    • clear - clears the content of the pop-up window
  • Managing visibility
    • show - shows the pop-up window
    • hide - hides the pop-up window
    • isVisible - checks whether the pop-up window is hidden
  • Handling events
    • attachEvent - attaches an event
    • detachEvent - detaches the specified event handler
  • Others
    • getItemData - get items of the list attached through attachList method
    • setSkin - sets the skin to apply
    • unload - destructor. Destroys a dhtmlxPopup object and clears the memory
