API Reference

showMonth(date)

Makes the specified date is currently visible in the calendar.

Parameters:

  • date - the date(s) you want to show. Should be specified as a Date object.
//shows June 2011 in the calendar
myCalendar.showMonth(new Date(2011,5,1));
