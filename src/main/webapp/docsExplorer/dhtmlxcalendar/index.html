dhtmlxCalendar

dhtmlxCalendar is a component that can take the form either of a popup datepicker or a flat calendar. Being a smart datepicker solution for users' websites or applications, this component is really easy to use and fully customizable to fit everyone's needs. Complete JavaScript API is provided to help users to save time on configuring.

Main features

  • Full control with JavaScript API;
  • Built-in multilingual support;
  • Easy connection to text input field;
  • Various skins;
  • Configurable data and view formats;
  • Custom holidays can be defined.
