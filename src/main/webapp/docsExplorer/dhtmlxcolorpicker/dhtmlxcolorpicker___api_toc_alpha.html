dhtmlxcolorpicker

dhtmlxColorPicker API Methods

  • attachEvent - attaches an event handler to ColorPicer
  • close - closes ColorPicker
  • detachEvent - removes an event handler
  • getSelectedColor - gets selected color
  • hide - hides ColorPicker
  • init - object initialization
  • linkTo - links color picker to some area
  • loadUserLanguage - sets new language interface for colorpicker
  • setColor - selects defined color
  • setCustomColors - defines user's custom colors
  • setImagePath - sets path to images
  • setOnCancelHandler - sets function called when selection is canceled
  • setOnSelectHandler - sets function called when color is selected (before closing)
  • setPosition - shows ColorPicker
  • show - shows ColorPicker
