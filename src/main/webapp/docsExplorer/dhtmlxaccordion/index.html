dhtmlxAccordion

dhtmlxAccordion is a component that provides an easy and effective way of placing different types of content on a single page. It's a very flexible and easy-to-use content manager, compatible with other dhtmlx components that can be attached to dhtmlxAccordion without any difficulties (supports only vertical layout).
